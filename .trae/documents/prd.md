## 1. 产品概述

智慧体育移动 App 是一款面向业余跑团教练与队员的综合训练管理平台，旨在通过数字化手段解决跑团训练计划发布难、执行跟踪难、数据收集散等核心痛点。教练可一键发布训练课表并实时查看队员完成率，队员可按天执行训练、上传反馈、查看趋势和报名队内活动，实现跑团训练的全流程数字化闭环。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 教练 | 邀请码注册/管理员分配 | 发布训练计划、查看完成率、管理分组、推送课表、审核请假、录入成绩 |
| 队员 | 手机号注册 | 查看训练日历、执行训练打卡、上传反馈、查看数据趋势、报名活动 |

### 2.2 功能模块

1. **首页**：今日概览、快捷入口、本周训练进度、最新通知摘要
2. **训练**：训练日历、课表详情、配速记录、心率区间、疲劳自评、打卡签到
3. **队伍**：分组名单、队内排名、请假报备、活动报名
4. **路线**：路线收藏、路线列表、路线详情（距离/爬升/描述）
5. **数据**：个人目标、成绩录入、赛前测评、装备寿命、伤病提醒、趋势图表
6. **通知**：消息通知列表、课表推送提醒、活动通知
7. **个人**：个人资料、角色切换（教练/队员视图）、设置

## 3. 核心流程

### 3.1 教练发布训练计划流程

教练在训练模块创建训练计划 → 设置日期、训练内容、目标配速和心率区间 → 选择推送对象（全队/分组） → 系统推送课表通知到队员 → 队员在训练日历中查看 → 执行训练并打卡 → 教练查看完成率统计

### 3.2 队员训练执行流程

队员收到课表推送通知 → 查看今日训练内容 → 执行训练 → 记录配速和心率 → 完成疲劳自评 → 一键打卡签到 → 查看训练趋势数据

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深夜蓝黑 (#0F172A) 作为背景，活力荧光绿 (#22D3EE → #10B981) 作为强调色，传达运动科技感
- **辅助色**：热力橙 (#F97316) 用于警告/提醒，柔和灰 (#94A3B8) 用于次级信息
- **按钮风格**：圆角胶囊按钮，带渐变效果和微弱发光阴影
- **字体**：标题使用 Outfit（几何运动感），正文使用 DM Sans（清晰可读）
- **布局风格**：移动端底部Tab导航，卡片式内容布局，圆角16px
- **图标风格**：线性描边图标（Lucide），2px 描边宽度
- **动画**：页面切换滑动过渡，打卡按钮弹性缩放，进度环渐变填充动画

### 4.3 响应式设计

- 移动端优先设计，适配 375px-428px 宽度
- 底部Tab导航栏固定5个入口（首页/训练/队伍/路线/数据），通知和个人通过顶部图标进入
- 触摸优化：按钮最小点击区域44px，滑块控件加大触控范围
- 卡片内容超长时内部滚动，页面整体不横向溢出

### 4.4 设计参考

整体风格参考 Nike Run Club + Strava 的深色运动科技风，强调数据可视化与即时反馈，通过荧光渐变色传达速度与活力。
